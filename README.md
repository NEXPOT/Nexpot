## 📌 프로젝트 소개
<p> 2022 한국관광공사 x Kakao 관광데이터 활용 공모전에서 입상(장려상)한 프로젝트입니다. </p> 
<p>
NEXPOT은 YouTube에서 본 여행지 정보를 한곳에 담은 Mash-up 서비스입니다. </br> 
여행지를 생생하게 보여주는 영상과 함께 상세 정보, 여행 코스, 지도를 제공합니다. </br>
수많은 리뷰를 시각화한 인포그래픽으로 여행지 리뷰를 한눈에 파악할 수 있습니다. </br>
</p>
